# Helmet Security

**Category:** Funded startup — MCP discovery and monitoring
**Funding:** $9M (December 2025) — SYN Ventures, WhiteRabbit Ventures
**Source:** https://www.securityweek.com/helmet-security-emerges-from-stealth-mode-with-9-million-in-funding/

## What It Does

Discovers, monitors, and manages MCP servers. Integrates into existing EDR and API tooling. Real-time monitoring of MCP communications for exfiltration and injection attempts.

## Coverage

Discovery, real-time monitoring, access control.

## What It Does NOT Cover

Content trust provenance classification.
