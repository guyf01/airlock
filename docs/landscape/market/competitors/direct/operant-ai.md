# Operant AI

**Category:** Funded startup — runtime AI agent security
**Funding:** $13.5M Series A (September 2024)
**Recognition:** Featured in all 5 Gartner AI security reports as of 2026
**Source:** https://www.operant.ai/solutions/mcp-gateway

## What It Does

Runtime inspection across MCP, Kubernetes, AWS Bedrock, Azure, Google Vertex. Blocks prompt injection, data exfiltration, and shell execution in real time. Maps high-risk data flows. Claims to detect "Shadow Escape zero-click exploits."

## Coverage

Threat detection across multiple surfaces, access control, runtime blocking.

## What It Does NOT Cover

Content trust provenance classification. Detection is pattern-based — block known threats, pass everything else. Does not classify responses by data source origin.
