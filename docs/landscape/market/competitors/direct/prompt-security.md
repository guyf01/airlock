# Prompt Security

**Category:** Acquired — now part of SentinelOne
**Funding pre-acquisition:** $23M ($5M seed + $18M Series A)
**Acquired by:** SentinelOne (2025/2026)
**Source:** https://prompt.security/blog/security-for-agentic-ai-unveiling-mcp-gateway-mcp-risk-assessment

## What It Does

Endpoint-level monitoring and control of MCP traffic. Discovers shadow MCP servers. Blocks malicious prompts and unauthorized actions. Now integrated into SentinelOne's platform.

## Coverage

Discovery, threat detection, endpoint monitoring.

## What It Does NOT Cover

Content trust provenance classification.

## Significance

The SentinelOne acquisition means endpoint security incumbents now have a dedicated MCP gateway product. Endpoint security vendors — already deployed in enterprise environments — gaining MCP-specific capabilities compresses the enterprise sales window for standalone tools.
