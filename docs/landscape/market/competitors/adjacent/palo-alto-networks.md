# Palo Alto Networks — Prisma AIRS

**Category:** Established security incumbent — enterprise security platform
**Product:** Prisma AIRS 2.0 (launched October 2025)
**Source:** https://www.paloaltonetworks.com/company/press/2025/palo-alto-networks-secures-the-ai-agent-revolution-with-the-launch-of-prisma-airs-2-0

## What It Does

Prisma AIRS 2.0 integrates the acquired Protect AI. In November 2025, added integration with Factory (an AI coding tool) that explicitly claims prompt injection inspection on tool calls in developer coding workflows.

Unit 42 (Palo Alto's threat research team) published an MCP attack vector taxonomy — signals active research investment in this space.

## Coverage

Enterprise security platform, prompt injection detection in developer coding context, runtime blocking, threat research.

## What It Does NOT Cover

Content trust provenance classification. Detection is pattern-based, not provenance-based.

## Relationship to Airlock

The Factory integration is the most direct enterprise incumbent overlap — same developer coding workflow, explicit claim of prompt injection inspection on tool calls. Palo Alto's enterprise distribution and brand make them a formidable presence in the security team buyer conversation.
