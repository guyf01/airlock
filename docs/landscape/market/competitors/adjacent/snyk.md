# Snyk

**Category:** Established security incumbent — developer security platform
**Move:** Acquired Invariant Labs (creators of mcp-scan) in June 2025
**Distribution:** 4,500+ enterprise customers, >25% Fortune 500
**Source:** https://snyk.io/news/snyk-acquires-invariant-labs-to-accelerate-agentic-ai-security-innovation/

## What It Does (MCP-Specific)

**Evo by Snyk** (October 2025): Agentic security orchestration with a dedicated MCP Scan Agent that discovers and assesses MCP servers and agent tool chains. Identifies vulnerabilities in MCP server packages.

**Secure At Inception** (August 2025): MCP-integrated real-time scanning at point of code generation in AI coding assistants. Sits in the developer coding workflow.

**mcp-scan** (from Invariant Labs): Static scanner and active proxy. In proxy mode, enforces guardrail rules (PII, secrets, custom policies), detects rug-pull attacks by hashing tool descriptions. Academic paper published at ACM.

## Coverage

Discovery, static scanning, runtime guardrails, developer workflow integration, enterprise distribution.

## What It Does NOT Cover

Content trust provenance classification.

## Strategic Risk

Developer-first positioning matches Airlock's go-to-market. Enterprise distribution is the gap — Snyk has it, Airlock does not. If Snyk adds provenance classification (they now have the research team who built the leading MCP security tool), the differentiation window closes.
